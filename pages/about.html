About This Project

Project Overview

This website is part of a bachelor thesis project. It presents research findings and analysis across five main chapters, each containing detailed information accessible through interactive popups.

Purpose

The purpose of this project is to present academic research in an accessible and interactive format, making complex information easier to understand through structured content and visual elements.

Structure

The website is organized into several sections:

  • Introduction: Overview of the research
  • Five main chapters: Detailed analysis and findings
  • Interactive popups: Additional information and details
  • Thesis: Summary and conclusions

Technical Details

This website is built using:

  • HTML5 for structure
  • CSS3 for styling
  • JavaScript for interactivity
  • Responsive design for all devices
